# pgen
A simple password generator  

## How to use  

simply  
```sh
pgen
```

The default lenght is 20 characters but you can change it with  
```sh
pgen -l 20
```

If you want to chose an alphabet  
```sh
pegn -alfa "AaBbCcDd123#!="
```
> **Note**  
> The alphabet is case sensitive,  
> so 'A' is different from 'a'
